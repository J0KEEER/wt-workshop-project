# Technology Stack

## Frontend (Client)
- **Framework**: React 18.3.1 (Vite-powered)
- **State Management**: React Hooks (Native)
- **Navigation**: React Router DOM 6.26.2
- **Styling**: Standard CSS (Custom Variables likely)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Networking**: Axios 1.7.7
- **Real-time**: Socket.io-client 4.8.3
- **Testing**: Jest, React Testing Library

## Backend (Server)
- **Runtime**: Node.js
- **Framework**: Express 4.21.0
- **ORM**: Sequelize 6.37.3
- **Database**: SQLite3 (Local file-based)
- **Authentication**: JWT (Jsonwebtoken 9.0.2), Bcryptjs 2.4.3
- **Validation**: Express-validator 7.2.0
- **Real-time**: Socket.io 4.8.3
- **Parsing**: xlsx (Excel parsing)
- **Environment**: Dotenv

## Development Tools
- **Build System**: Vite (Frontend)
- **Package Manager**: NPM
- **Testing Runner**: Jest
